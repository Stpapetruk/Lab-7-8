# Лабораторная работа: Связанный список процессов PCB


## Описание проекта

Проект представляет собой реализацию системы управления списком блоков
управления процессами с использованием односвязного списка на языке C++. 
Данная лабораторная работа демонстрирует практическое применение структур
данных для решения задач системного программирования включаяя управление
процессами в операционной системе

## Структура файлов

process-list.h - заголовочный файл с объявлением классов и структур
process-list.cpp - файл с реализацией методов классов
main.cpp - демонстрационная программа с тестовыми сценариями
CMakeLists.txt - конфигурационный файл для сборки проекта
.gitignore - для исключения временных файлов
README.md - документация проекта


## Функциональность программы
Программа демонстрирует:
Создание списка процессов
Добавление нескольких процессов
Вывод списка после добавленияУдаление процессов
Обработку ошибок 
Финальный вывод состояния списка

## Технические особенности
1) Баланс между простотой реализации и эффективностью операций поиска
2) Поддержание списка в отсортированном состоянии по ID процесса
3) Автоматическое управление памятью через конструкторы/деструкторы

## Сборка проекта
mkdir build
cd build
cmake ..
make
./process-list

## Требования к системе
Компилятор C++ стандарта C++17 или новее
CMake 3.10 или выше
